## VPinFE Release Notes

### Summary
This is the offical release of vpinfe v1.0.0.  Remember if you download the "slim" version you need to have "chrome" installed on your local machine.

### What's New
- DOF Support for both the frontend doing random effects and supporting pulling automated updates from the DOF Online Config Tool
- Ability to change from default vpinball version to another version on a per table basis.
